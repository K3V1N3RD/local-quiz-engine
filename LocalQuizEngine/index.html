Welcome

LocalQuizEngine is an offline quiz creation and study tool.

This application is a static web application built with HTML, and JavaScript. Despite being displayed in a web browser, it does not require an internet connection and is designed to operate entirely offline.

LocalQuizEngine does not connect to servers, transmit data, collect analytics, perform update checks, display advertisements, or use telemetry.

All quiz data remains on your device unless you choose to manually share exported quiz files.

Features include creating quizzes, loading quizzes, taking quizzes, reviewing results, and importing or exporting quizzes as JSON files.

About LocalQuizEngine

LocalQuizEngine is an offline quiz creation and study tool designed to be simple, fast, and transparent.

What Is a Static Web Application?

LocalQuizEngine is a static web application. In practical terms, this means the application consists entirely of local files that run inside your web browser. 

index.html
creator.js
quiz.js
validator.js
example-quiz.json

The browser loads the local files directly and executes the application entirely on your device.

Offline-Only Design

LocalQuizEngine is intentionally designed to operate completely offline. The app contains no designed mechanism for transmitting information. LocalQuizEngine...

  • Does not connect to the internet
  • Does not contact servers
  • Does not upload data
  • Does not download content
  • Does not check for updates
  • Does not display advertisements
  • Does not collect analytics
  • Does not use telemetry
  • Does not require accounts or sign-ins

No information is transmitted over a network by the application itself.

All quiz data remains on your device unless you deliberately export a quiz file and share it manually.

Security and Privacy

Because LocalQuizEngine operates entirely offline, it runs in a highly isolated environment. Meaning the security boundary is no longer "Can the application be trusted?" but rather "Can the device and browser be trusted?".

The application does not depend on cloud services, online databases, user accounts, or remote servers. As a result, there are no application-operated systems that can collect quiz data, usage information, scores, or personal information.

While no software can guarantee absolute security, the offline-only design greatly reduces the attack surface commonly associated with web services. There are no servers to compromise, no accounts to steal, and no network communications generated by the application.

Any security considerations are generally limited to the device on which the application is running and any files the user chooses to share manually.

Features

  • Create quizzes
  • Edit existing quizzes
  • Load quizzes
  • Take quizzes
  • Review results
  • Import quiz files
  • Export quiz files

Quiz files are stored in a human-readable JSON format that can be created either through the graphical interface or by manually editing the file structure.

Design Goals

  • Offline-first operation
  • Simplicity
  • Transparency
  • Fast response times
  • Minimal resource usage
  • Human-readable data formats

The application avoids artificial delays, mandatory animations, loading screens for local operations, and other interface behaviors that can make study tools feel slow or unresponsive.

User actions are designed to feel immediate and predictable.

Data Storage

LocalQuizEngine does not maintain cloud storage, user profiles, score history, or quiz history.

Quizzes exist as local JSON files that the user chooses to create, load, edit, export, and share.

The application does not automatically distribute, synchronize, back up, or transmit quiz data.
